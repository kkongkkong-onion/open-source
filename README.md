## 1. top
시스템의 실시간 성능을 모니터링하는 도구. 
- 현재 실행 중인 프로세스들의 목록, CPU 및 메모리 사용량 등을 실시간으로 표시함.

 주요 옵션
* d SECONDS: 갱신 주기를 초 단위로 설정함
* p PID: 특정 PID의 프로세스만 모니터링함
* u USER: 특정 사용자의 프로세스만 모니터링함
  주요 키
* q: top 명령어를 종료
* h: 도움말을 표시
* k: 프로세스를 종료 (PID를 입력해야 함)
* r: 프로세스의 우선순위를 변경
